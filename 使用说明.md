# 足球比赛预测工具 - 使用说明

## 功能概述

本工具基于 Rotowire.com 的数据源，结合天气、历史战绩、球员状态、裁判、赔率等多种因素，使用AI大模型进行足球比赛预测。

## 数据来源

- **比赛信息**: Rotowire.com (阵容、球员、教练、裁判)
- **天气数据**: OpenWeatherMap API
- **赔率信息**: 综合赔率数据
- **AI分析**: 支持OpenAI兼容接口的大模型

## 快速开始

### 方法一：使用EXE（推荐）

1. 双击 `足球比赛预测工具.exe`
2. 程序启动后，在配置区域填入模型信息
3. 点击"获取模型"按钮获取可用模型列表
4. 从下拉框选择模型
5. 左侧列表会显示未来3天的比赛
6. 选择一场比赛，等待详细信息加载
7. 点击"开始预测分析"

### 方法二：Python运行

```bash
pip install -r requirements.txt
python main.py
```

## 配置说明

### 归纳模型配置（用于信息整理）

| 配置项 | 说明 |
|--------|------|
| API地址 | 归纳模型的API地址 (如 https://api.openai.com) |
| API密钥 | 归纳模型的API密钥 |
| 模型 | 从下拉框选择 (推荐 gpt-3.5-turbo) |

### 分析模型配置（用于深度预测）

| 配置项 | 说明 |
|--------|------|
| API地址 | 分析模型的API地址 |
| API密钥 | 分析模型的API密钥 |
| 模型 | 从下拉框选择 (推荐 gpt-4) |

### 天气配置（可选）

| 配置项 | 说明 |
|--------|------|
| 天气API密钥 | OpenWeatherMap的API密钥 |

**免费获取天气API密钥：** https://openweathermap.org/api

## 预测分析内容

AI会基于以下信息进行分析：

### 比赛信息
- 比赛时间、地点、联赛
- 场地条件、容量

### 球队信息
- 教练信息（姓名、国籍、执教风格）
- 近期状态（最近5场战绩）
- 球队统计数据（进球、失球、控球率等）
- 预计首发阵容和阵型
- 伤病情况
- 关键球员

### 裁判信息
- 姓名、国籍
- 执法经验
- 场均出示黄牌数
- 场均判罚犯规数

### 赔率信息
- 胜负平赔率
- 大小球赔率
- 双方进球赔率
- 亚盘信息

### 天气信息
- 温度、湿度、风速
- 天气状况
- 对比赛的影响分析

### 历史交锋
- 总交锋次数
- 胜负平比例
- 最近5次交锋比分
- 场均进球数

## 分析输出

AI会提供以下分析：

1. **实力对比分析** - 双方近期状态、攻防数据对比
2. **战术分析** - 阵型、战术打法、克制关系
3. **关键因素分析** - 伤病、裁判、天气、心理因素
4. **数据统计分析** - 历史交锋、赔率概率、趋势
5. **风险提示** - 不确定性因素、冷门可能性
6. **最终预测** - 胜负概率、比分预测、大小球、双方进球

## 注意事项

1. **网络要求**：需要网络连接获取比赛信息
2. **API配额**：请确保大模型API有足够的配额
3. **天气数据**：不填天气API密钥将使用模拟数据
4. **预测仅供参考**：预测结果仅供参考，不构成任何投注建议
5. **信息加载**：选择比赛后需要几秒钟加载详细信息

## 故障排除

### 问题：测试连接失败
- 检查API地址是否正确
- 检查API密钥是否有效

### 问题：获取模型列表失败
- 确认API地址和密钥正确
- 部分API可能不支持此功能，可以手动输入模型名称

### 问题：无法获取比赛信息
- 检查网络连接
- 程序会自动使用演示数据作为备用

### 问题：AI分析失败
- 检查模型名称是否正确
- 检查API配额是否充足
